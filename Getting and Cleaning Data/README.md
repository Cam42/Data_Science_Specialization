run_analysis.R :
  - Program which assumes it is in the same directory as the uncompressed zip file from 'https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip'.  Will take the training and testing X and y data from them.  It then combines these into 1 dataframe and removes all features from X which are not a mean or standard deviation.  This leaves 66 features from X and the activity type used to collect data from y.  The output is the file 'run_analysis.txt' which contains the average values for all 66 features and 6 activity type combinations.  This means 'run_analysis.txt' is composed of 396 labels combining each activity and feature along with the mean accross all datapoints invovling both.  The first 5 feature and activities are: "tBodyAcc_mean_X_WALKING" "tBodyAcc_mean_Y_WALKING" "tBodyAcc_mean_Z_WALKING" "tBodyAcc_std_X_WALKING" "tBodyAcc_std_Y_WALKING".

CodeBook.md :
  - A code book describing the data obtained, transformations to obtain the mean of standard deviation/mean data for every unique activity + volunteer combination (180 total), and features of the final datastructure.
